# TP-Link Kasa Smart Plug

This module controls TP-Link Kasa Smart Plugs. It can control both single plug and multi plug devices.

Actions:

- Power On/Off/Toggle

Variables:

- SW Version
- HW Version
- Model
- Device ID
- OEM ID
- HW ID
- RSSI
- Latitude
- Longitude
- Alias
- MAC Address
- Power State
- Total Plugs

Feedbacks:

- Power State

Presets:

- Power On/Off/Toggle
